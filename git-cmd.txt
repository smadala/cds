======================
Branching
++++++++++++++++++++++


1)
$git branch
to show all the branches
* before name shows current working branch

2)
$git branch <new_branch_name>
to create new branch

3) 
$git checkout <branch_name>
to change branch name

================================
Tag
------++++++++++++++++++++++------

git tag -a v1.0 -m "release version on v1.0 ready"

git commit -a -m "message"

$git tag 
to show all tags

$git show v1.0
to show details of particular tag

$git push "name" v1.0
to push tag to remote repository

$git push "name" --tag
to push all the tags
